    6. meta tag

    tag is used to represent the metadata about HTML document. it specifies the page description, keywords, copyright, language, author of the documents

    meta data doesn't dsiplay on web page but it is used by search engines, browsers and other web services

    meta tag is placed within the head tag, it can be used more than one times in a document

    meta charset='utf-8'

    defines character encoding, which means it will support to display any language

    meta name='keyowrds' content='HTML,CSS JavaScript'

    it specifies the list of keywords whichs is used by search engines

    meta name='description' content='free online tutorial'

    it defines the website description which is useful to provide relevant search performed by search engines

    meta name='author' conent='thisauthor'

    it specifies the author of the page. It is useful to extract to extract author information by content management system automatically

    meta name='refresh' content='50'

    it specifies to provide instruction to the browser to automatically refresh the content after every 50sec

    meta http-equiv='refresh' content='5;url=htttp://www.google.co.in

    setted url with content so it will automatically redirect to the given page after the provided time

    meta name='viewport' content='width=device-width, intial-scale=1.0'

    viewport to control the page dimension and scaling so that our website looks good on all devices

    if this tag is present,it indicates that this page is mobile device supported
